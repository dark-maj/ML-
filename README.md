# Machine Learning Coursework - IIIT Hyderabad

This repository documents my journey through the Machine Learning coursework at IIIT Hyderabad, focusing on both theoretical understanding and practical implementation.

Key highlights:
- Implementation of ML algorithms from scratch
- Comparative analysis of different preprocessing techniques
- Experiments with data augmentation and normalization
- Performance evaluation across multiple models and datasets

The objective is to develop a deep understanding of Machine Learning concepts by building and testing models in real-world scenarios.
